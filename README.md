# Customer Support - Real-Time Video Support Platform

CustomerSupport is a video support platform. It lets an agent create a support session, share an invite link with a customer, run a real-time audio/video call, chat, share files, record, view history, and monitor live metrics.

## Stack

- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Realtime: Socket.IO
- Video: WebRTC with mediasoup SFU and mediasoup-client
- Uploads: Multer
- Recording: FFmpeg integration
- Charts: Chart.js
- Auth: JWT and bcryptjs

## Quick Start

```bash
npm install
npm start
```
